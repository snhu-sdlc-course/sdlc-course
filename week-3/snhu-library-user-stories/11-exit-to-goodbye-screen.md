# #11 Exit to Goodbye Screen

## User Story

> AS A user
>
> I WANT TO exit the application
>
> SO THAT I can safely terminate my user session and log out.

## Acceptance Criteria

* [ ] Given I am on the main screen, when I open the _File_ menu and click _Exit_, then the application logs me out and shows me a _Goodbye_ screen.
